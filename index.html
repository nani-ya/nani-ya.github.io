VSCode 自定义模板

VSCode中自定义模板，让写代码事半功倍

在Vue项目中,我们可以利用VSCode的snippet在.vue文件创建后能轻松地生成一套模板,使得新建.vue文件更加方便。

具体步骤如下

1、选择vscode中的 “ 文件 –> 首选项 –> 用户代码片段 ”,此时,会弹出一个搜索框,输入vue,选择 vue.json 进行编辑

2、在内容区中输入如下代码片段:

{
"Print to console": {
"prefix": "v3",
"body": [
"<template>",
" <div>$0</div>",
"</template>\n",
"<script type=\"text/javascript\">",
"export default {",
" name: \"app\",",
" setup(props.context) {",
" return {\n",
" }",
" },",
"}",
"</script>\n",
"<style lang=\"stylus\" scoped>",
"</style>",
"$2"
],
"description": "Log output to console"
}
}

3、内容介绍:

3.1、”prefix”: “v3”, 后面内容为自定义代码片段名称,例如:输入 “v3”+tab 键即可实现生成代码
3.2、”body” 中为代码片段的具体内容,由于是 vue3.0 的代码,所以和 vue2.0 在部分代码上有区别

hexo 部署方法

hexo部署：

hexo generate : 生成静态文章，可以用 hexo g 简写形式
hexo deploy : 部署文章，可以用hexo d简写形式
